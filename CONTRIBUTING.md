
# Contributing to NovaDeFi

Welcome! NovaDeFi is part of the Stellar Wave Program. Here's how to get started:

## How to Contribute

1. Browse the [Issues](../../issues) tab
2. Pick an issue tagged `good first issue`, `medium`, or `high`
3. Comment on the issue to let the maintainer know you're working on it
4. Fork the repo and create a branch: `git checkout -b fix/issue-name`
5. Make your changes and write tests
6. Submit a Pull Request with a clear description

## Code Style
- Follow standard Rust formatting: `cargo fmt` before every commit
- Run `cargo clippy` and fix all warnings
- Every contract function must have a corresponding test
